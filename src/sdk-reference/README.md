# SDK Reference

This directory contains the complete SDK reference for the Chutes SDK. Each file documents the classes, functions, and methods available in the SDK.

## Core Classes

- [Chute Class](/docs/sdk-reference/chute) - The main class for defining AI applications
- [Cord Decorator](/docs/sdk-reference/cord) - HTTP API endpoint decorator
- [Job Decorator](/docs/sdk-reference/job) - Background job decorator
- [Image Class](/docs/sdk-reference/image) - Docker image building
- [NodeSelector Class](/docs/sdk-reference/node-selector) - Hardware requirements

## Template Functions

- [Template Functions](/docs/sdk-reference/templates) - Pre-built template builders
